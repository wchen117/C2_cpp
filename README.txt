This code utilize the Jason parser for C++, which can be found at

JsonCpp
https://github.com/open-source-parsers/jsoncpp

1. Upon compilation, modify the INCL and LIB path to include the header and library of JsonCpp
2. add -std=c++11 (mac) or -std=c++0x(linux) after g++ command
3. use #include<json.h>
